# AR-Foundation – Juego 3D

Proyecto académico desarrollado con **Unity (C#)** utilizando **ARFoundation** como introducción a la realidad aumentada aplicada a videojuegos.

## Descripción
El proyecto consiste en un juego 3D en realidad aumentada donde el jugador debe **esquivar objetos** controlando el personaje mediante el **movimiento de la cabeza** (izquierda, derecha y salto).

## Funcionalidades principales
- Control del player mediante movimiento de cabeza.
- Desplazamiento lateral y salto.
- Detección de colisiones con obstáculos.
- Integración de ARFoundation.

## Tecnologías utilizadas
- Unity
- C#
- ARFoundation
- Desarrollo 3D

## Aprendizajes
- Uso de ARFoundation en proyectos interactivos.
- Programación de lógica de gameplay en entornos AR.
- Diseño de controles alternativos basados en movimiento.
